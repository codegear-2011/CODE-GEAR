Every CSS rule consists of two main parts ‚Äî a selector and a declaration block. The selector targets the HTML element you want to style, while the declaration block defines how that element should look.

CSS Rule Structure

Here‚Äôs the basic syntax of a CSS rule:

  property: value;

The selector specifies which HTML element to style. The declaration block (inside the curly braces) contains one or more declarations, each made of a property and a value.

Example

The following CSS rule makes all paragraph text red and center-aligned:

  color: red;
  text-align: center;
}

  • color ‚Üí Property, red ‚Üí Value
