CSS Tutorial

CSS Introduction Series(I)

Learn the fundamentals of CSS and start styling your web pages like a pro.

By CODE GEAR · October 9, 2025

CSS stands for Cascading Style Sheets. It is a stylesheet language used to style and enhance website presentation. CSS is one of the three main components of a webpage, along with HTML and JavaScript. HTML adds Structure to a web page. JavaScript adds logic to it and CSS makes it visually appealing or stylish. It controls the layout of a web page i.e. how HTML elements will be displayed on a webpage. CSS was released (in 1996), 3 years after HTML (in 1993). The main idea behind its use is that it allows the separation of content (HTML) from presentation (CSS). This makes websites easier to maintain and more flexible.

What is CSS?

CSS controls the visual presentation of HTML elements, enabling colors, fonts, layout, spacing, and animations.

Visual Impact of CSS

Before CSS

Plain HTML structure — no styling applied.

After CSS

Styled and modernized with colors, layout, and fonts.

Improved Layouts

CSS makes layouts flexible and visually organized.

Professional Look

Modern websites depend on CSS for their visual appeal.

Enhanced User Experience

CSS enhances readability and user engagement.

CSS Syntax

A CSS rule has a selector and a declaration block:

selector {
  property: value;
  property2: value2;
}

Example:

p {
  color: blue;
  font-size: 16px;
}

Ways to Add CSS

1. Inline CSS:

<p style="color: red;">Inline CSS example</p>

2. Internal CSS:

<style> h1 { color: green; } </style>

3. External CSS:

<link rel="stylesheet" href="styles.css">

CSS Selectors

Selectors determine which elements are styled:

/* Element */ h1 { color: purple; }
/* Class */ .highlight { background-color: yellow; }
/* ID */ #main { font-size: 24px; }
/* Group */ h1, h2, p { font-family: Arial; }

Box Model

Every element has content, padding, border, and margin:

div {
  width: 200px;
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}

Conclusion

CSS is essential for web design, enabling responsive layouts, beautiful typography, and interactive pages. Master these basics to build visually engaging websites.
